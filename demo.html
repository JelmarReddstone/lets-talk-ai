How It Works

AgentReady sits between your existing site and AI crawlers. No redesign needed — it intercepts agent requests and serves optimized content.

┌─────────────────┐ │ AI Agent │ │ GPTBot / Claude │ │ / PerplexityBot │ └────────┬────────┘ │ request with bot UA or ?agentready=1 │ ┌────────▼────────┐ │ AgentReady │ │ ┌────────────┐ │ │ │ Detector │ │ ← identifies AI agents │ │ Extractor │ │ ← strips DOM to clean content │ │ Schema │ │ ← adds JSON-LD structured data │ │ llms.txt │ │ ← site overview for LLMs │ └────────────┘ │ └────────┬────────┘ │ choice of: markdown / JSON / llms.txt │ ┌─────────────┼─────────────┐ ▼ ▼ ▼ ┌────────────┐ ┌──────────┐ ┌────────────┐ │ Markdown │ │ JSON-LD │ │ llms.txt │ │ Clean text │ │ Schema.org│ │ Site index │ │ + metadata │ │ Rich data │ │ for LLMs │ └────────────┘ └──────────┘ └────────────┘

Detect

Recognizes 40+ AI crawlers by user-agent. Auto-refreshes the list from community sources every 24 hours, with a bundled fallback.

Extract

Strips navigation, ads, scripts, cookie banners, and visual clutter. Pulls out the actual content, metadata, images, and links.

Structure

Wraps content in Schema.org JSON-LD and clean markdown with YAML frontmatter. Agents get machine-perfect data without guessing.

Serve

Routes agent requests to optimized endpoints. Human visitors see your normal site — agents see the clean version. Zero visual impact.

Features

Everything you need to make your site the preferred source for AI-powered discovery.

🔍

AI Agent Detection

Identifies 40+ AI bots including GPTBot, ClaudeBot, PerplexityBot, Google-Extended, Bingbot, and more. Automatically refreshes from Dark Visitors API and community lists with a local bundled fallback.

📄

llms.txt Support

Generates standards-compliant llms.txt and llms-full.txt files — the AI equivalent of robots.txt. Gives LLMs a complete site overview in seconds without crawling every page.

🏗️

Schema.org JSON-LD

Auto-generates rich structured data: WebPage, RealEstateListing, Organization, BreadcrumbList, and FAQPage. Supports @graph merging for complex multi-type pages.

✂️

Content Extraction

Strips the DOM down to actual content. Removes nav, footer, ads, scripts, cookie notices, social widgets, and tracking pixels. Returns clean, readable markdown.

🔗

Dual Format Endpoints

Add ?agentready=1 to any URL for markdown, or ?agentready=json for structured JSON. Also responds to Accept: text/markdown headers automatically.

🏠

Real Estate Ready

Built-in support for real estate listing schemas with price, bedrooms, bathrooms, floor size, amenities, GPS coordinates, energy labels, and more — out of the box.

🔄

Auto-Updating Agent List

Fetches the latest AI crawler list from remote sources (Dark Visitors, ai-robots-txt), caches for 24 hours, and falls back gracefully through bundled JSON to a hardcoded minimum.

⚡

Zero Dependencies

Pure PHP 7.4+. No frameworks, no Composer packages, no npm builds. Single require autoloader, or activate the WordPress plugin and write zero code.

🔌

WordPress Plugin

Activate and go. Auto-configures from WP settings, registers /llms.txt routes, injects JSON-LD into <head>, adds REST API endpoints, and provides an admin settings page.

📡

Content Feed

Serves a JSON feed of all pages at /agentready-feed.json — an AI-native sitemap. Agents can discover and index your entire site efficiently from a single endpoint.

Any PHP site — one require 
// One line loads everything — no more require_once per file
require_once 'path/to/agentready/autoload.php';

use AgentReady\Config;
use AgentReady\AgentDetector;
use AgentReady\ContentExtractor;

// Load config
$config = Config::fromFile(__DIR__ . '/config.json');
$detector = new AgentDetector($config);

// Intercept AI agent requests
if ($detector->isAgent() || $detector->isAgentRequest()) {
    $extractor = new ContentExtractor($config);
    $data = $extractor->extract($html);
    $format = $detector->getRequestedFormat();

    if ($format === 'json') {
        header('Content-Type: application/json');
        echo json_encode($extractor->toJson($data));
    } else {
        header('Content-Type: text/markdown');
        echo $extractor->toMarkdown($data);
    }
    exit;
}

Why AgentReady?

SEO was about making Google find your pages. AEO (Agent Engine Optimization) is about making AI agents understand and recommend your content.

🏆 Be The Recommended Result

When an AI agent searches 50 property sites, the ones serving structured, clean data get ranked highest. AgentReady ensures your listings are easy to parse, compare, and recommend.

⚡ 10x Faster Agent Crawling

Instead of downloading 500KB of HTML, CSS, and JavaScript per page, agents get a 5KB markdown file with all the essential data. Faster crawls = more pages indexed = better coverage.

🎯 Zero Data Loss

AI agents often miss critical details buried in complex DOM structures. AgentReady extracts price, specs, features, and description into clearly labeled fields — nothing falls through the cracks.

📋 Standards-Based

Built on proven standards: Schema.org JSON-LD, the llms.txt proposal, and markdown — formats that every major AI system already understands. No proprietary lock-in.

🔒 No Impact on Human Users

Your website looks and works exactly the same for regular visitors. AgentReady only activates when an AI crawler is detected — completely invisible to humans.

🔌 Universal Compatibility

Works with WordPress, static PHP sites, Laravel, Symfony, or any PHP stack. Zero external dependencies. Requires only PHP 7.4+ with the DOM extension (included in 99% of PHP installs).

🌐 Future-Proof

The agent list auto-updates from Dark Visitors and community sources. As new AI crawlers emerge, AgentReady recognizes them automatically — you don't need to update your code.

🏠 Industry-Specific Schemas

Pre-built structured data generators for real estate (RealEstateListing), organizations, breadcrumbs, and FAQs. Easily extensible for any industry vertical.

📊 Agent Analytics Ready

Identifies exactly which AI vendor (OpenAI, Anthropic, Google, Meta, etc.) and category (AI crawler, AI search, AI assistant) is accessing your site. Know who's reading your content.

What's In The Box

Six modular PHP classes, a single-file autoloader, a WordPress plugin, and a bundled agent database. No build step required.

Project Structure

│   ├── Config.php            # JSON config loader with dot-notation access
│   ├── AgentDetector.php     # Detects AI bots from UA / Accept / query param

│   ├── ContentExtractor.php  # HTML → clean markdown / structured JSON
│   ├── StructuredData.php    # Schema.org JSON-LD generator
│   └── LlmsTxt.php          # llms.txt / llms-full.txt generator
├── data/
│   └── known-agents.json    # Bundled database of 40+ AI crawlers
├── autoload.php              # Single require — loads all classes
├── agentready-wp.php         # WordPress plugin (activate & go)
├── config.example.json       # Example configuration
└── composer.json             # PSR-4 autoloading (optional)
